📋 1. Project Planning & Management

1.1 Project Proposal

Overview

This project delivers an end-to-end medical decision support system for heart disease risk prediction and analysis. It integrates four pillars of data science: Data Engineering (SQL Star Schema), Data Analysis (Excel), Data Visualization (Power BI), and Machine Learning with Explainable AI (Python, LightGBM, SHAP, Streamlit).

Objectives

  1. Build a binary classification model to predict heart disease risk using the CDC BRFSS dataset (319,795 patients, 48 features).
  2. Engineer 25+ advanced features from raw data (BMI categories, Risk Scores, Lifestyle Scores, Geographic Enrichment).
  3. Optimize model performance using Bayesian Hyperparameter Tuning (Optuna) achieving 0.8404 ROC-AUC and 80.57% Recall.
  4. Provide Explainable AI (XAI) using SHAP values so doctors can understand model decisions.
  5. Design a Star Schema data warehouse (6 dimension tables + 1 fact table) feeding into Power BI dashboards.
  6. Deliver an interactive Streamlit web application for real-time patient risk prediction.
